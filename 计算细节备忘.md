# 氧化还原电位预测：计算细节备忘

> 本文件由 Coder 生成，整理自 redox_potential 项目技术细节

## 体系构建

```
电解液盒（~3nm³，~500-1000 溶剂分子）
    +
两块电极（Au(111) 或 Pt，~2-3层原子）
    +
Li⁺ + 阴离子（PF6⁻ 或 FSI⁻）
    +
目标分子（EC, DMC, 添加剂等）
```

力场：OPLS-AA，所有参数在 `ff/` 目录的 XML 文件里。

## 常电位 MD 的实现

OpenMM 不原生支持常电位，有两套实现路径：

**路径 A：`ConstantPotentialForce`（如果可用）**
- 直接约束电极电势，体系自动调整表面电荷
- 电荷响应通过迭代求解 Laplace 方程

**路径 B：电荷拟合法**
```
每 N 步：
    1. 从当前分子偶极矩估算电极应带电荷 Q
    2. 把 Q 加到电极原子上
    3. 继续 MD
```
本项目用 `OPLSSimulation` 的实现，即路径 B。

## RedoxMC 的能量项

每个电荷态的总能量：

```
E_total(s) = E_intrinsic(s)           ← 气相电子亲和能，从 DFT 或文献来
            + E_electrode(s)          ← -z × F × V（电势能）
            + PE_solv(s)              ← 来自 OpenMM NonbondedForce
            + ΔG_solvation(s)         ← 溶剂化自由能修正
```

Metropolis 接受概率：
```
P_accept = min(1, exp(-ΔE / kT))
where ΔE = E_total(to) - E_total(from)
```

## 电子亲和能的来源

**两个选项：**

**① 直接从文献/数据库输入**（当前 workflow）
- 在 `configs/redox/config_redox.json` 里写死
- 适合：有实验数据或高精度量子化学结果

**② 从 PySCF DFT 计算**（DFT 接口 workflow）
```
单分子 → 优化几何构型 → 做基态/激发态能量计算
       → 算 electron affinity (EA) 和 ionization potential (IP)
       → 填进 RedoxParameters
```

gpu4pyscf 后端支持 GPU 加速，适合算一批分子。

## 电压扫描范围

通常扫 **-2.0 V 到 0.0 V vs vacuum**（对应 ~-0.6 V 到 +1.4 V vs Li/Li⁺）。

每步：
- equilibration：5000 步（~10 ps）
- production sampling：5000 步（~10 ps）
- RedoxMC：每帧跑 200-2000 步 MC

总耗时：单个电势点 ~30-60 ns 模拟时间（GPU 上）。

## 输出：模拟 LSV 曲线

```
V (vs vacuum)   fraction_reduced   simulated_current
    -2.0              0.98              ~0
    -1.8              0.90              0.4
    -1.6              0.60              1.5
    -1.4              0.25              -1.8  ← 峰
    -1.2              0.08              -0.6
    -1.0              0.02              ~0
```

电流代理 = `d(fraction_reduced) / dV`（数值微分）

峰位置即为 **half-wave potential E₁/₂**。

## 常见追问 Q&A

**Q: 为什么用中性盐单元而不是单离子？**
> 单离子溶剂化能取决于参考电极，无法直接比较。中性单元（Li⁺ + anion）避免基准问题。

**Q: soft-core 势是做什么的？**
> 去耦合（charging）时， LJ 势能在端点有奇异性。Soft-core 在 λ→0/1 时把 LJ 势能平滑掉，保证数值稳定。

**Q: 为什么需要平底约束（flat-bottom restraint）？**
> 去耦合过程中，静电消失后阴阳离子会飘散。约束把距离限制在 [0.25, 0.60] nm 内，保证所有 λ 窗口比较的是同一个几何定义。

**Q: 算出来和实验差几十 mV 怎么办？**
> 先检查：力场参数、溶剂化能校正、参考电极偏移。差异在 50 mV 内算可接受；更大则可能是缺少反应通道或 DFT 参数不准确。

**Q: 常电位 MD 的 pe_solv 为什么重要？**
> 同一个分子，溶剂化壳层不同（Li⁺ 配位、靠近电极与否），pe_solv 可以差几十 kJ/mol，直接影响 redox 电位精度。MD 采样了这种真实构型涨落。

**Q: 为什么每种分子独立跑 RedoxMC？**
> 电子转移是局域事件，主要取决于目标分子周围的溶剂壳层。多分子耦合会指数级增加态空间。独立 RedoxMC 是平均场近似，适合配方级筛选。
